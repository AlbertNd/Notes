# Laravel Debugbar

[Documentation](https://github.com/barryvdh/laravel-debugbar)

1. Installation 
    - `composer require barryvdh/laravel-debugbar --dev`
2. Une fois installer il y a un petit boutton qui apparit en bas a droite de la page. 
    - Un click pour faire apparaitre la barre contenant des informations:
        - Timeline : La partie du code qui a pris le plus de temps 
        - Exceptions : pour montrer les erreurs 
        - view : Les view inclus dans la page 
        - Route : les routes 
        - Queries : les requetes SQL 
        - Models : Les models utilis√©
        - Gate 
        - session
        - Request : information sur les request
    - L'iconne du petit dossier permet de montrer les requetes precedentes
